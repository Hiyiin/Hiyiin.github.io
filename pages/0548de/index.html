# 计算属性-computed

# 计算属性简述

  • 定义：计算属性是指要用的属性不存在，要通过已有的属性计算得来

  • 原理：底层借助了Object.defineproperty方法提供的getter和setter

  • get()执行事件：

    • 初次读取时执行一次
    • 当依赖的数据改变时执行一次

  • 优势：与methods相比，内部有缓存机制（复用），效率更高，调试方便。

    • methods实现

    • 计算属性实现

  • 备注：

    • 计算属性最终会出现在vm上，直接读取使用即可
    • 如果计算属性要被修改，那必须写set函数响应修改，且set中要引起计算时依赖的数据发生变化

# 姓名案例-计算属性实现

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>姓名案例_计算属性实现</title>
    <script src="../js/vue.js"></script>
    <style>
        *{
            margin-bottom: 20px;
        }
    </style>
</head>
<body>
    <div id="app">
        姓：<input type="text" v-model:value="fName"><br>
        名：<input type="text" v-model:value="lName"><br>
        姓名：<span>{{fullName}}</span><br>
        姓名：<span>{{fullName}}</span><br>
        姓名：<span>{{fullName}}</span><br>
    </div>
</body>
<script>
    Vue.config.productionTip = false; //阻止vue在启动时生成生产提示
    const vm = new Vue({
        el:'#app',
        data(){
            return {
                fName:"张",
                lName:"三"
            } 
        },
        computed:{
            fullName:{
                //get作用：当有人读取fullName时，get就会被调用,且返回值就是fullName的值
                get(){
                    console.log("@use");
                    return this.fName+'-'+this.lName;
                },
                set(value){
                    console.log('set',value);
                    const arr = value.split("-");
                    this.fName = arr[0];
                    this.lName = arr[1];
                }
            }
        },
        methods:{
            
        },
    });
</script>
</html>
